Space Train Schedule

Real time train schedule application

database

back-end web development

Front-End Web Development

TOOLS: Bootstrap 4, Firebase, MomentJS

The Space Train application is a live scheduling application. The user enters the train's name, destination, departure time and departure freqency into the form and the information is appended to the schedule. The application also calculates the next train's arrival time.

Enter details about your train and departure.

The application will calculate when the next train is coming and how far away it is. All data will be appended to the table of all trains.
